## 💎 Ventajas Competitivas (Oportunidades de Negocio)

Este software destaca por funcionalidades de alto valor añadido que permiten una monetización premium:

### 🚀 POS Ultra-Rápido y Resiliente
- **Operación Ininterrumpida**: El único POS que permite seguir vendiendo durante caídas de internet con sincronización automática posterior.
- **Captura Global Inteligente**: Integración nativa con lectores de barra que acelera la atención al cliente en horas punta.
- **Venta Pesable**: Soporte directo para balanzas y productos a granel.

### 🧠 Inteligencia Financiera y Proyecciones (Premium)
- **Proyección de Inventario**: Algoritmo que calcula el valor potencial de venta de todo el stock actual, permitiendo prever ingresos futuros.
- **Reporte Tributario Automatizado**: Cálculo automático de IVA Débito y Crédito sincronizado con compras y ventas.
- **Análisis de Utilidad Potencial**: Visualización clara del margen de beneficio esperado antes de realizar la venta.

### 📦 Gestión de Inventario y Recetas
- **Escalabilidad de Producción**: Sistema de recetas que descuenta insumos automáticamente, evitando mermas y errores humanos.
- **Conversión Inteligente**: Manejo automático de unidades (Kg, g, L, ml) para una precisión total en costos.

## ⚙️ Flujos de Trabajo Operativos

Para mantener la calidad y agilidad del desarrollo, utilizamos comandos especializados:

- `/run-app`: Inicia el entorno de desarrollo completo.
- `/sync-docs`: Sincroniza la documentación tras hitos importantes.
- `/test-e2e`: Ejecuta pruebas críticas de extremo a extremo para asegurar que el POS nunca falle en producción.
- `pnpm gen:types`: Sincronización segura de la base de datos con el código TypeScript.

## 🏗 Stack Tecnológico Moderno

- **Frontend**: Next.js 16 (App Router) & React 19 para un rendimiento máximo.
- **Estética**: Shadcn/UI con estándares de diseño premium (Modo oscuro adaptable, Micro-animaciones).
- **Backend & Seguridad**: Supabase con **Row Level Security (RLS)** para máxima protección de datos del cliente.
- **Gestor**: `pnpm` para instalaciones rápidas y deterministas.
