description: Inicia el entorno de desarrollo completo con múltiples terminales.
---

Este flujo guía el arranque del proyecto para máxima visibilidad en los terminales del IDE.

> [!TIP]
> Ejecuta cada comando en un terminal independiente para ver los logs en tiempo real.

1. **Terminal 1**: Iniciar servidor de desarrollo.
```bash
npm run dev
```

2. **Terminal 2**: Terminal de trabajo para comandos adicionales.
```bash
# Espacio para git, migraciones, etc.
```

3. **Terminal 3**: CLI de Gemini para asistencia persistente.
```bash
npx gemini init
```
